# Transmitly.ChannelProvider.Mailgun.Configuration

A shared configuration for the Transmitly Mailgun channel provider.

### Getting started

If you're just starting out with Transmitly, you likely want to be using the [transmitly-channel-provider-Mailgun](https://github.com/transmitly/transmitly-channel-provider-mailgun) package instead.

See the [Transmitly](https://github.com/transmitly/transmitly) project for more details on what a channel provider is and how it can be configured.
